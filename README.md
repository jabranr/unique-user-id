# Unique User ID

Generate a unique ID using the UNIX timestamp and random numbers.

## Usage

Install package

```sh
npm install unique-user-id
```

Use it generate an ID

```js
import uniqueUserId from 'unique-user-id';
```

## Run tests

Requires node v20 test runner.

```
npm t
```
